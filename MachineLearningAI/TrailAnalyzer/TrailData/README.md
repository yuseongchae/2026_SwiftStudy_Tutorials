# CoreML: Train a Core ML model

Discover how easy it is to train a machine learning model with Create ML. Using provided data, train a model to predict how challenging a trail is based on characteristics like distance and elevation.

## Overview

These resources are associated with the [Train a Core ML model](https://developer.apple.com/tutorials/develop-in-swift/train-a-core-ml-model) tutorial. This is part of the [Machine learning and AI](https://developer.apple.com/tutorials/develop-in-swift/welcome-to-machine-learning-and-ai) tutorials from [Develop in Swift](https://developer.apple.com/tutorials/develop-in-swift) by Apple.

## Using these resources

Follow the tutorial instructions in section one of [Train a Core ML model](https://developer.apple.com/tutorials/develop-in-swift/train-a-core-ml-model) to use these resources within the Xcode app.
