# Grateful Moments: Collect, model, and store data

Start building an app that lets people record moments of gratitude. In this tutorial, you’ll create an entry view with a photo picker. You’ll also model your data and save information for each entry.

In the next tutorial, you’ll display entries in a list view and show each entry’s details in a detail view.

## Overview

These resources are associated with the [Collect, model, and store data](https://developer.apple.com/tutorials/develop-in-swift/collect-model-and-store-data) tutorial. This is part of the [App Development](https://developer.apple.com/tutorials/develop-in-swift/welcome-to-app-development) tutorials from [Develop in Swift](https://developer.apple.com/tutorials/develop-in-swift) by Apple.

## Using these resources

Follow the tutorial instructions in section one of [Collect, model, and store data](https://developer.apple.com/tutorials/develop-in-swift/collect-model-and-store-data) to use these resources within the Xcode app.
